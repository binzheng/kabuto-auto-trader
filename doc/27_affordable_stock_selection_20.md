## 📝 よくある質問（FAQ）

### Q1: なぜ50万円以下に限定？

**A:** 少額運用での資金効率最大化のため

```
理由:
1. 口座100-300万円で10銘柄以上保有可能
2. 分散効果を実現
3. 高額株は資金を大きく占有
4. ポジション調整の柔軟性

例:
  口座300万円で10銘柄保有（平均30万円）
  vs
  口座300万円で2銘柄のみ（キーエンス700万+α不可）
```

### Q2: NTT（1.2万円）は本当に推奨？

**A:** 初心者・少額運用には有用だが、注意点あり

**メリット:**
- 最少資金で購入可能
- 配当利回り高い（3-4%）
- 超ディフェンシブ

**デメリット:**
- ボラティリティ極低（ATR 2-5円）
- トレンドフォロー戦略に不向き
- スプレッドの相対的影響大

**結論:**
- 口座100万円: 分散目的で有用 ○
- 口座200万円以上: 不要 ×
- Phase 3では除外推奨

### Q3: 商社4銘柄は多すぎない？

**A:** 確かに同一セクター集中のリスクあり

**対策:**
```
同時保有ルール:
  商社は最大2銘柄まで

推奨組み合わせ:
  - 三菱商事（8058）+ 三井物産（8031）
    理由: 業態・資源分散

避けるべき:
  - 商社4銘柄全て保有
    理由: 資源価格リスク集中
```

### Q4: ENEOSは本当に推奨？

**A:** 中級者以上、原油市場理解者向け

**特徴:**
- 原油価格と高相関
- ボラティリティ高い（2-4%）
- 景気敏感

**推奨条件:**
- 原油市場の理解がある
- ボラティリティ許容できる
- 口座資金200万円以上

**初心者は除外推奨**
